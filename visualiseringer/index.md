# Visualiseringer

## naa-situasjon-arkitektur.html
- **Beskrivelse**: ArchiMate-inspirert diagram som viser dagens arkitektur for helsedatadeling i Norge. Inkluderer forretningslag (aktorer), applikasjonslag (nasjonale e-helselosninger, EPJ-systemer, stottetjenester), teknologilag (helsenettet, sikkerhet) og internasjonalt grensesnitt (MyHealth@EU pilotstatus). Interaktive tooltips med detaljert informasjon per komponent. Viser sentrale dataflyter og nokkelutfordringer.
- **Sist oppdatert**: 2026-03-15

## ehds-maalarkitektur.html
- **Beskrivelse**: ArchiMate-inspirert diagram som viser EHDS-malarkitekturen for Norge basert pa forordning (EU) 2025/327. Inkluderer styrings- og forvaltningslag, forretningslag med utvidede rettigheter, applikasjonslag delt i primaer- og sekundaerbruk, EU-infrastruktur (MyHealth@EU og HealthData@EU), og teknologilag med nye standarder (EEHRxF). Viser prioriterte helsedatakategorier med tidsfrister, sentrale dataflyter og implementeringstidslinje fra 2025-2035. Komponenter merket som eksisterende, ma tilpasses, ny eller EU-komponent.
- **Sist oppdatert**: 2026-03-15

## gap-analyse-per-alternativ.html
- **Beskrivelse**: Visuell sammenligning av tre EHDS-tiltaksalternativer langs fem arkitekturkvalitetsdimensjoner (interoperabilitet, teknisk gjeld, vendor lock-in, skalerbarhet, sikkerhet by design). Inkluderer radardiagram, stolpediagram for samlet score og integrasjonskompleksitet, detaljert heatmap per dimensjon med underkategorier, og komponentpåvirkningsoversikt for eksisterende og nye komponenter. Responsivt design med Chart.js. Basert pa rapporter/arkitekturkonsekvenser-ehds.md.
- **Sist oppdatert**: 2026-03-15

## tidslinje-ehds-implementering.html
- **Beskrivelse**: Interaktiv HTML-tidslinje som viser EHDS-implementeringsplanen for Norge fra 2026 til 2035. Inkluderer EU-frister (2027, 2029, 2031), norske milepæler for EØS-innlemmelse, lovendringer og teknisk utvikling, og parallelle spor for juridisk arbeid, organisatorisk etablering, primærbruk, sekundærbruk og FHIR-overgang. Hover/klikk for detaljer om hver milepæl. Selvstendige HTML med CSS/JS. Basert på rapporter/anbefaling-ehds.md og rapporter/gjennomfoeringsforutsetninger-ehds.md.
- **Sist oppdatert**: 2026-03-15

## risikomatrise-ehds.html
- **Beskrivelse**: Interaktiv risikomatrise med 15 identifiserte risikoer for EHDS-implementering i Norge. Risikoer plottet på sannsynlighet (1-4) x konsekvens (1-4) med fargekoding (grønn/gul/oransje/rød). Kategorimerking (juridisk, organisatorisk, teknisk, økonomisk, ekstern). Klikk for detaljer om hver risiko inkludert beskrivelse og risikoreduserende tiltak. Inkluderer oppsummeringskort og sortert risiko-liste. Selvstendige HTML med CSS/JS. Basert på rapporter/gjennomfoeringsforutsetninger-ehds.md og rapporter/anbefaling-ehds.md.
- **Sist oppdatert**: 2026-03-15

## ehds-utredning-dashbord.html
- **Beskrivelse**: Overordnet dashbord for ledelsen med nøkkeltall (kostnad, tidsfrister, antall gap, antall risikoer), anbefalt strategi i kortversjon med tre faser, samsvarsstatus per område (rettslig, teknisk, organisatorisk, semantisk) med fargekoding og stolpediagram, radardiagram for arkitekturkvalitet per alternativ, tidslinjeoversikt med EU-frister og norske milepæler, top 5 risikoer med tiltak, og kostnadsprofil for hybridstrategien. Selvstendige HTML med CSS/JS og Chart.js. Basert på rapporter/ehds-utredning-samlet.md.
- **Sist oppdatert**: 2026-03-15

## aktoer-ansvar-oversikt.html
- **Beskrivelse**: Visualisering av hvem gjør hva i EHDS-innføringen. Viser seks aktører (HOD, Helsedirektoratet, NHN, RHF-ene, KS/kommuner, EPJ-leverandører) med roller (strategi, lovgivning, drift, etterlevelse, utvikling), konkrete oppgaver per fase, tidshorisont per aktør og FHIR-modenhet for EPJ-leverandører. Inkluderer ansvarsmatrise (RACI-inspirert) for ni funksjonsområder. Selvstendige HTML med CSS/JS. Basert på rapporter/ehds-utredning-samlet.md og rapporter/gjennomfoeringsforutsetninger-ehds.md.
- **Sist oppdatert**: 2026-03-15
